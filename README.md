### Install Guides:
Pre-Requisites:
  - python3 installed 
  - git installed

In your terminal, type:
```
git clone git@github.com:mabdullah32/CJWM.git
```
```
cd CJWM
```
```
pip install -r requirements.txt
```


### Launch Codes:
```
python app/__init__.py
```
